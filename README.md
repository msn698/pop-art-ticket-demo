# POPICON Ticket Page Demo

A custom-coded ticket booking page demo inspired by bold exhibition-style interactions.

## Features
- Animated hero with counters
- Scrolling ticker strip
- Ticket tier selection with quantity controls
- Live order summary and total
- WhatsApp checkout link with selected tickets

## Configure
Set WhatsApp number in `script.js` (`WHATSAPP_PHONE`).

## Deploy
Ready for GitHub Pages from `main` branch root.
